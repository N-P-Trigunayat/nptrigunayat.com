DanceVibe Studio: Classes Timetable + Booking Site

Overview

DanceVibe Studio offered multiple dance styles but communicated everything via Instagram stories, causing students and parents to constantly miss updates. They wanted a dedicated place where people could view schedules, book trials, and understand what each class offers.

The Problem

The studio had no consistent timetable, trial bookings got lost in chats, and instructors had no record of attendance. Parents often asked the same questions because the information lived only in social media posts.

What I Built

I built a lively, modern dance website that matched the studio's energy:

  • Weekly class timetable with levels and instructors
  • Stylish instructor profiles
  • Trial booking system with instant notifications
  • Attendance panel for instructors
  • Polished service descriptions for each style

How It Works

Visitors browse the timetable, choose a class, and book a trial. The instructor gets notified and can prepare for the new student. Attendance marking takes seconds, making batch management cleaner.

Impact

The studio saw more trial bookings, fewer repetitive queries, and more structured batch distribution. Parents appreciated clarity. Instructors finally had a simple way to track attendance.

Key Outcomes

  • Improved student experience
  • Better batch control
  • More professional studio identity
  • Fewer scheduling issues
